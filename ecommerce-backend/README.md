# Ecommerce Backend

This is the Node.js backend for the ecommerce project.

## What It Does

The backend handles:

- products
- cart items
- delivery options
- orders
- payment summary data
- reset/default data loading

It uses Express and Sequelize, and it loads default data the first time it runs.

## Main Folder

- `routes/` contains the API routes.
- `models/` contains the database models.
- `defaultData/` contains the starter data.
- `images/` contains the static images used by the app.

## Run It

1. Install Node.js 22 or newer.
2. Open this folder in VS Code.
3. Run `npm install`.
4. Run `npm run dev`.

## Useful Command

- `npm run zip`
